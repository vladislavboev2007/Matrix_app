**MatriX** — это интерактивное веб-приложение-тренажер для изучения и отработки навыков работы с матрицами. Разработано для учащихся 9–11 классов, студентов и всех, кто готовится к экзаменам (ОГЭ, ЕГЭ, вступительные испытания).

## 🚀 Возможности

### 👤 Авторизация и профиль
- Ввод имени, фамилии и класса (9, 10, 11)
- Автоматическое распознавание при повторном входе
- Персонализированный интерфейс

### 📚 Форматы тестирования
| Формат | Описание |
|--------|----------|
| **Готовые тесты** | Три уровня сложности (лёгкий, средний, сложный) |
| **Типовые задания** | Выбор конкретной операции и количества вопросов (5, 10, 15) |
| **Контрольная работа** | Случайная генерация 15–20 вопросов по всем темам с оценкой |

### ✏️ Типы заданий
- **Выбор ответа (MCQ)** — классические тесты с вариантами
- **Заполнение пропуска** — ввод числового ответа
- **Верно/Неверно (True/False)** — оценка истинности утверждения

### 📊 Поддерживаемые операции с матрицами
| Класс | Операции |
|-------|----------|
| **9 класс** | Сложение матриц, умножение на скаляр, транспонирование, определение размера |
| **10 класс** | Умножение матриц, определители 2×2 и 3×3, обратная матрица, ранг, системы линейных уравнений (Крамер, Гаусс) |
| **11 класс** | Миноры, алгебраические дополнения |

### ⏱️ Таймер и навигация
- Таймер обратного отсчета (от 7 до 60 минут)
- Визуальное предупреждение при окончании времени
- Кнопки навигации между вопросами
- Возможность отмены теста

### 📈 Статистика и отчеты
- Сохранение результатов в PostgreSQL
- История всех пройденных тестов
- Скачивание отчета в формате DOCX с подробным разбором ошибок

### 🌟 Задача дня
- Ежедневная генерация уникального задания (на основе seed от даты)
- Подсчет серии правильных ответов (streak)
- Типы задач: определитель 3×3, обратная матрица, умножение матриц, ранг, системы уравнений

### 📖 Теоретический раздел
- Интерактивное оглавление
- Подробное объяснение всех тем с формулами и примерами
- Навигация по разделам

## 🚀 Установка и запуск


1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/vladislavboev2007/Matrix_app.git
   cd Matrix_app
   ```

2. **Создание виртуального окружения**
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   
   # Linux/Mac
   source .venv/bin/activate
   ```
   
3. **Установка зависимостей**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройка базы данных PostgreSQL**
   
   4.1. Создаем базу данных в PostgreSQL
   ```sql
   CREATE DATABASE matrix_tests_db;
   ```
   4.2. Затем загружаем туда из `database/matrix_tests_db.sql` через формат Plain в окне Restore
   
5. **Настройка подключения к БД**
 
    Отредактируйте файл `backend/database.py`:

    ```python
    DATABASE_URL = "postgresql+asyncpg://matrix_user:your_password@localhost:5432/matrix_tests_db"
    ```
6. **Запуск приложения**

    ```bash
    cd backend
    uvicorn main:app --host 0.0.0.0 --port 8000 --reload
    ```

7. **Доступ к приложению**

    Откройте браузер и перейдите по адресу: http://localhost:8000
