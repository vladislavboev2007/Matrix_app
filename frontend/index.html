1. Что такое матрица

Матрица — это прямоугольная таблица чисел, расположенных в строках и столбцах. Матрица размера m×n имеет m строк и n столбцов.

A = | 1 2 3 | | 4 5 6 |

Это матрица размера 2×3 (2 строки, 3 столбца). Элемент на пересечении i-й строки и j-го столбца обозначается aij.

Виды матриц

  • Квадратная — число строк равно числу столбцов (n×n)
  • Прямоугольная — m×n, где m ≠ n
  • Нулевая — все элементы равны нулю
  • Единичная (E) — на главной диагонали 1, остальные 0
E = | 1 0 | E = | 1 0 0 | | 0 1 | | 0 1 0 | | 0 0 1 |
Запомни: Размер матрицы всегда записывается как «строки × столбцы».

2. Сложение матриц

Складывать можно только матрицы одинакового размера. Сложение выполняется поэлементно.

C = A + B, где cij = aij + bij

Пример

A = | 1 2 | B = | 5 6 | A+B = | 6 8 | | 3 4 | | 7 8 | | 10 12|

Свойства

  • Переместительность: A + B = B + A
  • Сочетательность: (A + B) + C = A + (B + C)
  • A + 0 = A (нулевая матрица — нейтральный элемент)
Ошибка: Нельзя складывать матрицы разных размеров — операция не определена.

3. Умножение матрицы на скаляр

Скаляр — это обычное число. При умножении матрицы на скаляр k каждый элемент умножается на k.

B = k · A, где bij = k · aij

Пример

A = | 1 2 | 3·A = | 3 6 | | 3 4 | | 9 12 |

Свойства

  • k·(A + B) = k·A + k·B
  • (k + m)·A = k·A + m·A
  • 1·A = A, 0·A = 0

4. Умножение матриц

Произведение A·B определено только если число столбцов A равно числу строк B. Если A имеет размер m×k, а B — k×n, то C = A·B имеет размер m×n.

cij = сумма(aik · bkj) по всем k

Пример (2×2)

A = | 1 2 | B = | 5 6 | | 3 4 | | 7 8 | C = A·B: c₁₁ = 1·5 + 2·7 = 19 c₁₂ = 1·6 + 2·8 = 22 c₂₁ = 3·5 + 4·7 = 43 c₂₂ = 3·6 + 4·8 = 50 C = | 19 22 | | 43 50 |
Важно: A·B ≠ B·A — умножение матриц не переместительно!

5. Транспонирование

Транспонированная матрица Aᵀ получается заменой строк на столбцы: строки исходной матрицы становятся столбцами транспонированной.

(Aᵀ)ij = Aji

Пример

A = | 1 2 3 | Aᵀ = | 1 4 | | 4 5 6 | | 2 5 | | 3 6 |

Матрица 2×3 стала матрицей 3×2.

Свойства

  • (Aᵀ)ᵀ = A
  • (A + B)ᵀ = Aᵀ + Bᵀ
  • (A·B)ᵀ = Bᵀ·Aᵀ
  • (k·A)ᵀ = k·Aᵀ

6. Определитель матрицы

Определитель — число, характеризующее квадратную матрицу. Обозначается det(A) или |A|.

Определитель 2×2

A = | a b | det(A) = a·d - b·c | c d |

Пример 2×2

A = | 3 2 | det(A) = 3·4 - 2·1 = 12 - 2 = 10 | 1 4 |

Определитель 3×3 (правило Саррюса)

A = | a b c | | d e f | | g h i | det(A) = a(ei - fh) - b(di - fg) + c(dh - eg)

Пример 3×3

A = | 1 2 3 | | 4 5 6 | | 7 8 9 | det(A) = 1·(5·9 - 6·8) - 2·(4·9 - 6·7) + 3·(4·8 - 5·7) = 1·(45-48) - 2·(36-42) + 3·(32-35) = 1·(-3) - 2·(-6) + 3·(-3) = -3 + 12 - 9 = 0
Важно: Если det(A) = 0, матрица называется вырожденной — обратной матрицы не существует.

7. Обратная матрица

Обратная матрица A⁻¹ — такая матрица, что A · A⁻¹ = A⁻¹ · A = E (единичная матрица). Существует только если det(A) ≠ 0.

Формула для матрицы 2×2

A = | a b | 1 | d -b | | c d | A⁻¹ = ————— · | | det(A) | -c a |

Пример

A = | 2 1 | det(A) = 2·4 - 1·3 = 5 | 3 4 | A⁻¹ = (1/5) · | 4 -1 | = | 0.8 -0.2 | | -3 2 | | -0.6 0.4 |

Проверка

A · A⁻¹ = | 2 1 | · | 0.8 -0.2 | = | 1 0 | = E ✓ | 3 4 | | -0.6 0.4 | | 0 1 |

8. Ранг матрицы

Ранг матрицы — максимальное число линейно независимых строк (или столбцов). Обозначается rang(A) или rk(A).

Метод Гаусса (элементарные преобразования)

  • Переставить строки/столбцы местами
  • Умножить строку на ненулевое число
  • Прибавить к одной строке другую, умноженную на число

После приведения к ступенчатому виду ранг = число ненулевых строк.

Пример

A = | 1 2 3 | → | 1 2 3 | → | 1 2 3 | | 2 4 6 | | 0 0 0 | | 0 1 2 | | 0 1 2 | | 0 1 2 | | 0 0 0 | rang(A) = 2 (две ненулевые строки)
Свойство: rang(A) ≤ min(m, n), где m — строки, n — столбцы.

9. Системы линейных уравнений

Система линейных уравнений (СЛУ) вида Ax = b решается несколькими методами.

Метод Крамера (для квадратных систем)

Если det(A) ≠ 0, система имеет единственное решение:

xi = det(Ai) / det(A)

где Ai — матрица, полученная заменой i-го столбца на столбец правых частей b.

Пример

2x₁ + x₂ = 5 x₁ + 3x₂ = 10 A = | 2 1 | b = | 5 | det(A) = 2·3 - 1·1 = 5 | 1 3 | | 10 | A₁ = | 5 1 | det(A₁) = 5·3 - 1·10 = 5 x₁ = 5/5 = 1 | 10 3 | A₂ = | 2 5 | det(A₂) = 2·10 - 5·1 = 15 x₂ = 15/5 = 3 | 1 10 | Ответ: x₁ = 1, x₂ = 3

Метод Гаусса

Записываем расширенную матрицу [A|b] и приводим к ступенчатому виду элементарными преобразованиями над строками, затем находим решение обратной подстановкой.

Теорема Кронекера–Капелли: СЛУ совместна тогда и только тогда, когда rang(A) = rang(A|b).

Метод Гаусса (прямой ход + обратная подстановка)

Записываем расширенную матрицу системы [A|b] и элементарными преобразованиями строк приводим к ступенчатому виду, затем находим решение обратной подстановкой.

Система: Расширенная матрица [A|b]: 2x₁ + x₂ = 5 | 2 1 | 5 | x₁ + 3x₂ = 10 | 1 3 | 10| Шаг 1: R2 = R2 − (1/2)·R1 | 2 1 | 5 | | 0 2.5 | 7.5 | Шаг 2 (обратная подстановка): 2.5·x₂ = 7.5 → x₂ = 3 2·x₁ = 5 − 1·3 = 2 → x₁ = 1 Ответ: x₁ = 1, x₂ = 3
Крамер vs Гаусс: Крамер — удобен для 2×2, 3×3 с единственным решением. Гаусс — универсален для любых систем.

10. Минор и алгебраическое дополнение — 11 класс

Минор Mij элемента aij матрицы A — определитель матрицы, полученной вычёркиванием i-й строки и j-го столбца из A.

Mij = det(A без строки i и столбца j)

Алгебраическое дополнение

Алгебраическое дополнение Aij — минор с учётом знака:

Aij = (−1)^(i+j) · Mij

Знак определяется шахматным правилом: позиции (+) и (−) чередуются:

Шахматная матрица знаков: | + − + | | − + − | | + − + |

Пример

A = | 1 2 3 | | 4 5 6 | | 7 8 9 | M₁₂ (вычёркиваем строку 1, столбец 2): M₁₂ = det| 4 6 | = 4·9 − 6·7 = 36 − 42 = −6 | 7 9 | A₁₂ = (−1)^(1+2) · M₁₂ = (−1)³ · (−6) = +6

Разложение по строке (теорема Лапласа)

Определитель можно вычислить как сумму произведений элементов любой строки на их алгебраические дополнения:

det(A) = ai1·Ai1 + ai2·Ai2 + ... + ain·Ain
Применение: Миноры и алгебраические дополнения используются при вычислении определителей матриц высокого порядка и нахождении обратной матрицы через матрицу алгебраических дополнений (союзная матрица).
